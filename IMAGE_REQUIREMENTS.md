# Image Requirements for Internee.pk Website

## Required Images and Their Locations

### 1. Hero Section Images
**Location:** `public/` directory

- **`startup-winners.png`**
  - Description: Image showing startup competition winners/awards
  - Size: Recommended 800x600px or similar aspect ratio
  - Used in: Hero section right column, top image card

- **`community-collage.png`**
  - Description: Collage of community members/students
  - Size: Recommended 800x600px or similar aspect ratio
  - Used in: Hero section right column, bottom image card

### 2. Partner/Logo Images
**Location:** `public/partners/` directory

All partner logos should be:
- Format: PNG with transparent background (for light mode) or colored (for dark mode grayscale effect)
- Size: Recommended 200x60px or maintain aspect ratio
- The component will automatically apply grayscale filter in dark mode

Required files:
- **`google-cloud.png`** - Google Cloud logo
- **`see-pakistan-2024.png`** - SEE Pakistan 2024 logo
- **`founders-hub.png`** - Microsoft for Startups Founders Hub logo
- **`itg.png`** - ITG logo
- **`nep-nics.png`** - National Expansion Plan of NICS logo
- **`pitb.png`** - PITB (Punjab Information Technology Board) logo

### 3. Internship Card Images
**Location:** `public/internships/` directory

These images are displayed in the 6 internship cards section on the homepage:
- **`1.png`** - Graphic Design internship image
- **`2.png`** - Chatbot Development internship image
- **`3.png`** - Other Internships image
- **`4.png`** - Frontend Internship image
- **`5.png`** - App Development internship image
- **`6.png`** - Backend Development internship image

**Size:** Recommended 400x300px or similar aspect ratio
**Format:** PNG or JPG

### 4. Component Icons
Most icons are currently using emoji or SVG. If you want to replace with actual image files:

- Professional Development section icons (Target, Mentorship, Certificate)
- Task Management Platform icons
- Tech Instructor section icons
- AI Mock Interviews section icons
- Career Resources icons (Resume, Mock Interviews, Skill Analysis, Learning Paths)

## Image Naming Convention
- Use lowercase with hyphens: `image-name.png`
- Keep filenames descriptive but concise
- Maintain consistent naming patterns

## Notes
- All images should be optimized for web (compressed but maintaining quality)
- Ensure images work well in both light and dark themes
- Partner logos will automatically be grayscale in dark mode
- Internship card images should be visually appealing and relevant to each category
