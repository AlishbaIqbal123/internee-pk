# Internship Category Images Requirements

## Location
All internship category images should be placed in: `public/internships/`

## Required Images

Based on the "Explore Internship Categories" section on the home screen, you need to add the following 6 images:

### 1. Graphic Design
- **Filename:** `1.png`
- **Location:** `public/internships/1.png`
- **Description:** Image representing graphic design work (e.g., design tools, creative workspace, design mockups)
- **Recommended Size:** 400x300px or similar aspect ratio

### 2. Chatbot Development
- **Filename:** `2.png`
- **Location:** `public/internships/2.png`
- **Description:** Image representing chatbot/AI development (e.g., chatbot interface, AI technology, conversation flow)
- **Recommended Size:** 400x300px or similar aspect ratio

### 3. Other Internships
- **Filename:** `3.png`
- **Location:** `public/internships/3.png`
- **Description:** Generic image representing various internship opportunities
- **Recommended Size:** 400x300px or similar aspect ratio

### 4. Frontend Internship
- **Filename:** `4.png`
- **Location:** `public/internships/4.png`
- **Description:** Image representing frontend development (e.g., code editor, browser, UI components)
- **Recommended Size:** 400x300px or similar aspect ratio

### 5. App Development
- **Filename:** `5.png`
- **Location:** `public/internships/5.png`
- **Description:** Image representing mobile app development (e.g., smartphones, app interfaces, mobile development)
- **Recommended Size:** 400x300px or similar aspect ratio

### 6. Backend Development
- **Filename:** `6.png`
- **Location:** `public/internships/6.png`
- **Description:** Image representing backend development (e.g., server, database, API, cloud infrastructure)
- **Recommended Size:** 400x300px or similar aspect ratio

## Image Specifications
- **Format:** PNG or JPG
- **Aspect Ratio:** 4:3 or 16:9 recommended
- **File Size:** Optimized for web (under 200KB per image recommended)
- **Quality:** High quality but web-optimized
- **Theme Compatibility:** Images should work well in both light and dark themes

## How Images Are Used
These images are displayed in the `InternshipCards` component on the home page. Each card shows:
- The image at the top
- The internship category name
- An "Apply Now" link

The images are automatically styled with hover effects and responsive sizing.

## Notes
- If an image is missing, a placeholder will be shown
- Images should be visually appealing and relevant to each category
- Ensure images are properly optimized for fast loading
- Consider using images that match the dark theme aesthetic of the website
